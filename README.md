# chatbot-com-gemini
Esse projeto tinha como objetivo aplicar os conhecimentos aprendidos durante as aulas sobre o framework flask e a aplicação de sqlite.
Esse é um chatbot que usa o Gemini como base, onde um arquivo de texto, fornecido na área de ADM, é usado como contexto para as perguntas do usuário.
Para seu funcionamento é necessário a configuração da variável de ambiente “GEMINI_API_KEY”
