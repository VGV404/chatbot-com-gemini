Ciência da Computação é o estudo dos princípios e técnicas que regem a manipulação de informações digitais. Isso inclui desde a teoria da computação e algoritmos até a concepção e implementação de software e hardware.
Os computadores funcionam através da execução de instruções codificadas em linguagem de máquina. Essas instruções são processadas pela CPU (Unidade Central de Processamento), que manipula dados armazenados na memória, seguindo as regras definidas pela arquitetura do computador.
Um algoritmo é uma sequência de passos finitos e bem definidos para resolver um problema específico ou realizar uma tarefa, geralmente expressa em forma de pseudocódigo ou linguagem de programação.
Hardware refere-se aos componentes físicos de um computador, como processador, memória, disco rígido etc. Já o software são os programas e dados que são executados no hardware, incluindo sistemas operacionais, aplicativos e jogos.
Programação é o processo de escrever e testar códigos para criar programas de computador que executam tarefas específicas ou resolvem problemas. Algumas das linguagens de programação mais comuns incluem Python, Java, C++, JavaScript, C#, Ruby, entre outras.
Inteligência artificial (IA) refere-se à capacidade das máquinas de realizar tarefas que normalmente exigiriam inteligência humana. Isso inclui aprendizado de máquina, processamento de linguagem natural, visão computacional, entre outras áreas. Os computadores aprendem por meio de algoritmos de aprendizado de máquina, que os capacitam a reconhecer padrões nos dados e ajustar seu comportamento de acordo com esses padrões. Aprendizado de máquina é uma subárea da inteligência artificial que se concentra no desenvolvimento de algoritmos que permitem aos computadores aprender a partir de dados, sem serem explicitamente programados.
Os computadores entendem a linguagem humana por meio de técnicas de processamento de linguagem natural (PLN), que envolvem o uso de algoritmos para analisar e interpretar textos escritos ou falados.
Criptografia é o processo de codificar informações de forma que apenas pessoas autorizadas possam acessá-las. Isso é feito através do uso de algoritmos matemáticos para embaralhar os dados, tornando-os ilegíveis para qualquer pessoa que não tenha a chave de decodificação correta.
Hacking refere-se a explorar sistemas de computador para ganhar acesso não autorizado ou realizar modificações não autorizadas. Os hackers podem invadir sistemas de computador por meio de várias técnicas, como exploração de vulnerabilidades de segurança, engenharia social, phishing, entre outras.
Um banco de dados é um sistema organizado para armazenar e gerenciar grandes volumes de dados, permitindo o acesso rápido e eficiente a essas informações. Os bancos de dados são usados para armazenar informações em uma estrutura organizada, permitindo que aplicativos e usuários realizem consultas e atualizações de forma eficiente.
Computação em nuvem é o fornecimento de serviços de computação, como armazenamento, processamento de dados e aplicativos, através da internet, em vez de serem executados localmente em hardware físico. Na computação em nuvem, os recursos de hardware e software são fornecidos sob demanda através da internet, permitindo que os usuários acessem e utilizem esses recursos conforme necessário.
Big Data refere-se a conjuntos de dados extremamente grandes e complexos que são difíceis de processar usando métodos tradicionais de processamento de dados. Grandes conjuntos de dados são processados utilizando técnicas avançadas de análise de dados, como computação distribuída, algoritmos de aprendizado de máquina e ferramentas de processamento paralelo, entre outras técnicas.
Realidade virtual (RV) é uma tecnologia que simula um ambiente tridimensional e interativo, proporcionando uma experiência imersiva ao usuário. A realidade virtual é criada por meio de dispositivos de hardware, como óculos de RV, e software especializado que gera e exibe ambientes digitais em tempo real. Realidade aumentada (RA) é uma tecnologia que combina elementos virtuais com o ambiente físico, permitindo que os usuários interajam com objetos digitais sobrepostos ao mundo real. Enquanto a realidade virtual cria um ambiente completamente digital e imersivo, a realidade aumentada superpõe elementos digitais ao ambiente físico, mantendo a interação com o mundo real.
Segurança cibernética é o conjunto de práticas, tecnologias e medidas de proteção utilizadas para proteger sistemas de computador, redes e dados contra-ataques cibernéticos e acessos não autorizados. Para proteger nossos computadores contra ameaças cibernéticas, podemos usar firewalls, software antivírus, manter os sistemas atualizados, criar senhas fortes e estar atentos a e-mails e links suspeitos.
Um vírus de computador é um tipo de malware projetado para se espalhar de um computador para outro e causar danos, como corromper dados, roubar informações ou controlar o sistema. Os vírus de computador podem se espalhar por meio de e-mails infectados, downloads de software malicioso, dispositivos USB contaminados e exploração de vulnerabilidades em sistemas operacionais e aplicativos.
Um firewall é um dispositivo de segurança que monitora e controla o tráfego de rede, bloqueando ou permitindo o acesso com base em regras de segurança predefinidas. Os firewalls protegem os computadores filtrando o tráfego de rede e bloqueando conexões suspeitas ou maliciosas, ajudando a prevenir ataques de hackers e malware.
Um sistema operacional é um conjunto de programas e recursos que gerenciam os recursos de hardware de um computador e fornecem serviços para aplicativos de software. Alguns dos sistemas operacionais mais populares incluem Windows, macOS, Linux, Android e iOS.
Um navegador da web é um aplicativo de software projetado para acessar e exibir páginas da web, permitindo aos usuários navegar na internet. Os navegadores da web funcionam interpretando e exibindo conteúdo HTML, CSS e JavaScript de páginas da web, permitindo aos usuários interagir com links, formulários e outros elementos.
Um site é uma coleção de páginas da web relacionadas, acessíveis através de um navegador da web e geralmente hospedadas em um servidor da internet. Os sites são criados por meio de linguagens de marcação, como HTML e CSS, e podem incluir scripts em JavaScript para interatividade. Eles são desenvolvidos usando editores de texto simples ou ambientes de desenvolvimento web integrados.
Codificação é o processo de escrever instruções precisas em uma linguagem de programação para criar programas de computador e aplicativos. As etapas para escrever um programa de computador geralmente incluem: entender os requisitos do projeto, projetar a lógica do programa, escrever o código fonte, compilar ou interpretar o código e testar e depurar o programa.
Um bug de software é um erro ou falha no código de um programa que faz com que ele se comporte de maneira inesperada ou produza resultados incorretos. Os bugs de software são corrigidos por meio de depuração, que envolve identificar, isolar e corrigir o código problemático. Isso é feito através do uso de ferramentas de desenvolvimento e técnicas de teste para encontrar e corrigir os erros no código.
Um smartphone é um dispositivo eletrônico que combina as funcionalidades de um telefone celular com recursos avançados, como acesso à internet, aplicativos, câmeras, entre outros. Os smartphones funcionam através de uma combinação de hardware e software, incluindo processadores, memória, sistema operacional e aplicativos, que permitem aos usuários realizar uma variedade de tarefas, como fazer chamadas, enviar mensagens, navegar na internet e executar aplicativos.
A Internet das Coisas (IoT) é uma rede de dispositivos físicos conectados à internet, que podem coletar e trocar dados entre si, bem como com sistemas de computador centralizados. A IoT está mudando o mundo ao possibilitar a automação de processos, otimização de recursos, melhoria da eficiência e criação de novos produtos e serviços conectados.
A robótica é um campo da engenharia e ciência que envolve o design, construção e operação de robôs, máquinas programáveis capazes de realizar tarefas de forma autônoma ou controlada por humanos. Os robôs são programados por meio de linguagens de programação específicas ou ambientes de desenvolvimento, que permitem aos programadores escrever instruções para controlar o comportamento e as ações do robô.
Uma rede de computadores é um conjunto de dispositivos interconectados que podem trocar dados e recursos entre si, permitindo comunicação e colaboração entre usuários e sistemas. As redes de computadores funcionam através da transmissão de dados entre dispositivos conectados, utilizando protocolos de comunicação e tecnologias de hardware, como cabos, roteadores e switches.
Um servidor é um computador ou sistema de computadores dedicado a fornecer serviços, recursos ou dados para outros dispositivos em uma rede. Os servidores são usados na computação para hospedar sites, aplicativos, armazenar dados, gerenciar redes, executar serviços de e-mail, entre outras funções de computação em rede.
Um algoritmo de busca é um conjunto de regras ou procedimentos utilizados para encontrar informações relevantes em um conjunto de dados, como páginas da web ou bancos de dados. Os algoritmos de busca funcionam examinando e classificando o conteúdo disponível com base em critérios como relevância, autoridade e popularidade, para fornecer resultados úteis aos usuários.
Um site de mídia social é uma plataforma online que permite aos usuários criar perfis, compartilhar conteúdo, interagir com outros usuários e participar de comunidades online. Os sites de mídia social funcionam reunindo usuários e conteúdo em uma plataforma online, onde podem interagir através de postagens, comentários, curtidas e compartilhamentos.
Comércio eletrônico é a compra e venda de produtos e serviços pela internet, utilizando plataformas online e sistemas de pagamento eletrônico. O comércio eletrônico funciona através de websites de comércio eletrônico, onde os consumidores podem navegar por produtos, fazer compras, realizar pagamentos online e receber entregas em casa.
Um aplicativo móvel é um software projetado para ser executado em dispositivos móveis, como smartphones e tablets, geralmente disponível para download em lojas de aplicativos. Os aplicativos móveis são desenvolvidos por meio de linguagens de programação específicas, como Java ou Swift, usando ferramentas de desenvolvimento de software e frameworks para dispositivos móveis.
Um banco de dados relacional é um tipo de banco de dados que organiza dados em tabelas com linhas e colunas, permitindo relações entre diferentes conjuntos de dados. Os bancos de dados relacionais funcionam armazenando dados em tabelas

A UniDrummond aderiu ao PROUNI – Programa Universidade para Todos. Uma iniciativa do Governo Federal que concede bolsas de estudos integrais e parciais aos alunos.
As bolsas do PROUNI valem para toda a duração do curso, mas para continuar recebendo o benefício o estudante precisa ser aprovado, no mínimo, em 75% das disciplinas cursadas em cada período letivo e seguir o regimento da Instituição.
As inscrições para o PROUNI são gratuitas e on-line, exclusivamente na página do programa na internet (siteprouni.mec.gov.br). Os interessados deverão se cadastrar no sistema com CPF e número de inscrição no Enem e poderão escolher até duas opções de bolsa, em ordem de preferência, sendo possível alterar as inscrições durante o período.
O Programa utiliza notas do Exame Nacional do Ensino Médio (Enem) como único critério para seleção dos candidatos para bolsas do ensino superior.
Para o PROUNI, podem participar somente aqueles que prestaram a última edição do ENEN e obtiveram nota média de 450 pontos nas provas objetivas, bem como pontuação maior que zero na redação.
Ainda para o PROUNI para os critérios de renda, o candidato deve comprovar renda familiar por pessoa de, no máximo, 1,5 salário-mínimo para concorrer a bolsas integrais. Já para as parciais metade da mensalidade, a renda familiar deve ser de até três salários-mínimos por pessoa, além de ter cursado o ensino médio completo na rede pública de ensino ou em escola particular na condição de bolsista integral.
O aluno também pode usar o FIES – Fundo de Financiamento Estudantil para você realizar seu sonho do ensino superior. Por meio do Programa, o aluno pode financiar seu curso superior na Drummond, integral ou parcial, com juros de apenas 6,5% ao ano.
